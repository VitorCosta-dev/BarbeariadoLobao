# Barbearia do Lobão Android

Projeto Android nativo para visualizar o protótipo como app.

## Como abrir

1. Abra o Android Studio.
2. Clique em `Open`.
3. Selecione a pasta `BarbeariaDoLobaoAndroid`.
4. Aguarde o Gradle sincronizar.
5. Clique em `Run` para abrir no emulador ou celular Android.

O app contém telas navegáveis de login, cadastro, página da barbearia, agendamento, pagamento, confirmação e gestão.
